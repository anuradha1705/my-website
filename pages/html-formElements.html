HTML Form Elements

HTML forms are made up of different form elements that allow users to input data. These elements include text fields, checkboxes, radio buttons, buttons, dropdowns, and more.

Common HTML Form Elements

  • <input>: Used to create various types of input fields like text, password, checkbox, radio, button, submit, file, etc.
  • <textarea>: For multi-line text input.
  • <select>: Dropdown list to select one or more options.
  • <option>: Defines options inside the select dropdown.
  • <button>: Creates clickable buttons (can submit form or run scripts).
  • <label>: Labels for inputs for better accessibility and usability.
  • <fieldset>: Groups related elements inside a form.
  • <legend>: Caption for the fieldset group.

Example: Multiple Form Elements

<form action="#" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" placeholder="Enter your name" required />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" placeholder="Enter your email" required />

  <label>Gender:</label>
  <input type="radio" id="male" name="gender" value="male" /> <label for="male">Male</label>
  <input type="radio" id="female" name="gender" value="female" /> <label for="female">Female</label>

  <label>Hobbies:</label>
  <input type="checkbox" id="reading" name="hobbies" value="reading" /> <label for="reading">Reading</label>
  <input type="checkbox" id="travel" name="hobbies" value="travel" /> <label for="travel">Travel</label>

  <label for="country">Country:</label>
  <select id="country" name="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
    <option value="uk">UK</option>
    <option value="australia">Australia</option>
  </select>

  <label for="message">Message:</label>
  <textarea id="message" name="message" rows="4" placeholder="Type your message here..."></textarea>

  <button type="submit">Submit</button>
</form>
