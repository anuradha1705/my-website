HTML Bootcamp

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages and web applications. HTML uses tags to structure content, add images, links, text formatting, and more.

Basic Structure of an HTML Document

Every HTML document has a basic skeleton:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Page Title</title>
  </head>
  <body>
    
  </body>
</html>

This structure ensures your page is correctly interpreted by browsers.

Commonly Used HTML Tags

  • <h1> to <h6> — Headings


  • <p> — Paragraphs


  • <a> — Hyperlinks


  • <img> — Images


  • <ul>, <ol>, <li> — Lists


  • <div> — Container for grouping content


  • <span> — Inline container


  • <form>, <input> — Forms and inputs


  • <table>, <tr>, <td> — Tables


How to Add Images

Use the <img> tag with src and alt attributes:


<img src="image.jpg" alt="Description of image" />


Creating Links

Use the <a> tag with the href attribute:


<a href="https://www.example.com">Visit Example</a>


Using Classes and IDs

Classes and IDs are attributes to identify and style elements with CSS or manipulate with JavaScript. 

<div class="container">Content</div>
<h1 id="main-heading">Title</h1>


Forms in HTML

Forms collect user input:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />
  <button type="submit">Submit</button>
</form>


Summary

This bootcamp covered the essential building blocks of HTML including structure, tags, attributes, images, links, classes, and forms. Mastering these fundamentals is key to web development success.

Quick Recap:
  • HTML structures web content using tags.

  • Every page has a <html>, <head>, and <body>.

  • Common tags: headings, paragraphs, links, images, lists, divs, spans.

  • Use attributes like src, href, class, and id to add more meaning.

  • Forms allow user interaction.
