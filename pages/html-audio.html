HTML Audio

The <audio> tag in HTML is used to embed sound content such as music or recordings on a web page. You can also control the audio with play, pause, and volume features.

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

üîç Explanation

  • <audio>: Main tag to insert audio.
  • controls: Adds audio player UI like play/pause/volume.
  • <source>: Defines audio file and format.
  • Multiple source files can be added for different formats.
  • Fallback text appears if browser doesn't support audio.
