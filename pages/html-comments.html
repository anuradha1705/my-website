HTML Comments

Comments in HTML are used to insert notes or explanations in the code which are not displayed on the web page.

They help developers understand the code better or temporarily disable parts of code without deleting them.

Syntax

The syntax for an HTML comment is:

<!-- This is a comment -->

<!-- <p>This paragraph is hidden by comment.</p> -->

Welcome to DevByAnu

This paragraph is visible.

Important Points

  • Comments are ignored by browsers and do not affect the display.
  • Comments start with <!-- and end with -->.
  • You cannot nest comments inside other comments.
  • Comments can span multiple lines.
  • Use comments to make code more readable and maintainable.
