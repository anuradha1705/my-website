HTML ID Attribute

The id attribute assigns a unique identifier to an HTML element.

What is the id attribute?

- It uniquely identifies one element on the page.
- Each id value must be unique within the HTML document.
- IDs are used to style specific elements with CSS or target them in JavaScript.
- IDs can also be used as anchor points for links within the page.

Syntax:

<tag id="uniqueID">...</tag>

Example:

<style>
  #special {
    color: white;
    background-color: #d6006f;
    padding: 10px;
    border-radius: 8px;
  }
</style>

<p id="special">This paragraph has a unique style using ID.</p>
<p>This paragraph is normal.</p>

Output:

This paragraph has a unique style using ID.

This paragraph is normal.

Using ID for Anchors:

Anchor links use IDs to jump to specific parts of a page:

<a href="#section1">Go to Section 1</a>

<h2 id="section1">Section 1</h2>
<p>Content for section 1.</p>

Output:

Go to Section 1

Section 1

Content for section 1.

Important Points:

  • The id value must be unique on a page (no duplicates allowed).
  • IDs are case-sensitive and should start with a letter.
  • IDs help select specific elements easily with CSS and JavaScript.
  • Use classes when multiple elements share the same style; use IDs for unique elements.
