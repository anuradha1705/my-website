HTML Basic

HTML (HyperText Markup Language) is the basic building block of the web. It uses elements and tags to structure and display content in a web browser.

Basic Elements of HTML

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a>
  • Images: <img>
  • Lists: <ul>, <ol>, <li>

Basic Syntax

Every HTML element has an opening tag, content, and a closing tag:

<p>This is a paragraph</p>

HTML Basic Page Example

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is a basic HTML page.</p>
    <a href="https://www.example.com">Visit Example</a>
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
    <img src="image.jpg" alt="Sample Image" width="200">
  </body>
</html>

Explanation

  • <!DOCTYPE html> - Declares the HTML version
  • <html> - Root tag
  • <head> - Metadata section
  • <title> - Page title shown in browser tab
  • <body> - Main visible content
  • <h1> - Main heading
  • <p> - Paragraph text
  • <a> - Hyperlink
  • <ul>, <li> - List and list items
  • <img> - Image with src and alt attributes
