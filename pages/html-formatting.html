HTML Text Formatting

HTML provides tags to format text and make it bold, italic, underlined, and more. These tags help to emphasize or decorate parts of your content.

Common Formatting Tags:

  • <b> – Bold text (without importance)
  • <strong> – Important bold text
  • <i> – Italic text (without importance)
  • <em> – Emphasized text (important italic)
  • <u> – Underlined text
  • <mark> – Highlighted text
  • <small> – Smaller text
  • <del> – Deleted (strikethrough) text
  • <ins> – Inserted (underlined) text
  • <sub> – Subscript (e.g., H2O)
  • <sup> – Superscript (e.g., 22)

Example:

<p>This is <b>bold</b> text.</p>
<p>This is <i>italic</i> text.</p>
<p>This is <u>underlined</u> text.</p>
<p>This is <mark>highlighted</mark> text.</p>
<p>Water formula: H<sub>2</sub>O</p>
<p>2<sup>3</sup> = 8</p>

Output:

This is bold text.

This is italic text.

This is underlined text.

This is highlighted text.

Water formula: H2O

23 = 8
