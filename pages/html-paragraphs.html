HTML Paragraphs

In HTML, paragraphs are defined using the <p> tag. Paragraphs are blocks of text used to describe or explain content. They automatically add space before and after the text.

Syntax:

<p>This is a paragraph.</p>

Example:

<html>
  <body>

    <p>This is my first paragraph.</p>
    <p>This is my second paragraph.</p>

  </body>
</html>

Explanation:

  • Each paragraph starts with <p> and ends with </p>.
  • Browsers automatically add space between paragraphs.
  • Paragraphs help organize content into readable sections.
  • You should not nest other block-level tags inside a paragraph.
