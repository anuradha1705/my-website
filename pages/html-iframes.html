HTML Iframes

The <iframe> tag is used to embed another HTML page within the current page. It creates a rectangular area that displays another webpage inside your page.

Key points about <iframe>:

  • An iframe embeds an external webpage or content within your page.
  • The src attribute specifies the URL of the page to embed.
  • You can control the size of the iframe using width and height attributes.
  • iframes allow you to include content like videos, maps, ads, or even another part of your site.

Syntax:

<iframe src="URL" width="width" height="height"></iframe>

Example:

<iframe src="file:///C:/Users/DELL/Desktop/website/pages/html-iframes.html" width="600" height="400"></iframe>

Attributes of <iframe>:

  • src: URL of the page to display inside the iframe.
  • width: Width of the iframe (in pixels or %).
  • height: Height of the iframe.
  • title: Accessibility title for the iframe content.
  • frameborder: Defines border around iframe (0 = no border).
  • allowfullscreen: Allows fullscreen mode for embedded videos.
