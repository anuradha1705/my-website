HTML File Paths

In HTML, file paths are used to tell the browser where to find files like images, stylesheets, or other documents (like PDFs). File paths can be **absolute** or **relative** depending on their location.

1. Absolute File Path

This is the complete URL of the file. It starts from the domain name.

<img src="https://www.example.com/images/pic.jpg" alt="Example Image">

2. Relative File Path

This is used when the file is inside your project folder. It is relative to the location of the current HTML file.

a) Same Folder

<img src="photo.jpg" alt="Photo">

b) Folder Inside Current Folder

<img src="images/pic.jpg" alt="Inside Folder">

c) Go One Folder Back

<img src="../pic.jpg" alt="Back One Folder">

d) Go Two Folders Back

<img src="../../pic.jpg" alt="Back Two Folders">
