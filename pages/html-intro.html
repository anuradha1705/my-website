HTML Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML defines the structure of a web page using various elements and tags.

What is HyperText?

HyperText means text that has links to other documents. When you click a link on a website, you are using hypertext.

What is Markup Language?

A markup language uses tags to describe elements. These tags tell the browser how to display content such as text, images, and links.

Key Features of HTML
