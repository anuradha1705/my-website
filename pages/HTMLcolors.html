HTML Colors

HTML colors are used to set the background color, text color, border color, etc. Colors can be defined using names, HEX codes, RGB, HSL, and more.

Ways to Define Colors

  • Color Names: red, blue, pink, etc.
  • HEX Codes: #ff0000 (Red), #00ff00 (Green)
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5) (with transparency)
  • HSL: hsl(0, 100%, 50%)

Syntax

<p style="color: red;">This is red text.</p>

<div style="background-color: #ffccff;">This has a light pink background.</div>

Example with Output

<p style="color: blue;">This text is blue.</p>
<p style="background-color: #f4c2c2;">This background is light red (HEX).</p>
<p style="color: rgb(0, 128, 0);">This text is green (RGB).</p>
<p style="color: rgba(255, 0, 0, 0.5);">This is semi-transparent red.</p>
<p style="color: hsl(300, 100%, 25%);">This text uses HSL color.</p>

Output:

This text is blue.

This background is light red (HEX).

This text is green (RGB).

This is semi-transparent red.

This text uses HSL color.

Explanation

  • style="color:" is used to change the text color.
  • style="background-color:" changes the background color.
  • HEX, RGB, RGBA, HSL give you more flexibility in choosing shades and transparency.
Summary:

HTML colors are used to design attractive web pages. You can define colors using names, HEX, RGB, RGBA, or HSL. You can apply them to text, backgrounds, borders, and more using the style attribute.
