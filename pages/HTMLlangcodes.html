HTML Language Code

The lang attribute in HTML is used to declare the **language** of the web page content. This helps browsers, screen readers, and search engines understand the correct language of your text.

Why Use lang Attribute?

  • Improves accessibility for screen readers.
  • Helps search engines understand your content language.
  • Improves SEO and translation tools support.

Syntax

<html lang="en">
  <head>
    <title>My Website</title>
  </head>
  <body>
    <p>Hello, World!</p>
  </body>
</html>

Common Language Codes

  • en - English
  • hi - Hindi
  • fr - French
  • es - Spanish
  • de - German
  • ja - Japanese

Output

Bonjour le monde!

नमस्ते दुनिया!

Hello World!

Summary:

The lang attribute in HTML defines the language of your content. This improves accessibility and helps search engines, translators, and screen readers. Use it in the <html> tag for best practice.
