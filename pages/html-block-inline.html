HTML Block and Inline Elements

In HTML, elements are mainly categorized as Block-level or Inline elements based on how they behave on the webpage.

Block-level Elements

Block-level elements take up the full width available, and start on a new line. They form the "blocks" of your webpage structure.

  • Examples: <div>, <h1> to <h6>, <p>, <section>, <article>
  • They always start on a new line.
  • You can set width, height, margin, and padding on them.

Example of Block-level elements:

<div>This is a block-level element.</div>
<p>Paragraph is also block-level.</p>
<h2>Headings are block-level.</h2>
This is a block-level element.

Inline Elements

Inline elements only take up as much width as necessary and do not start on a new line. They flow within the content.

  • Examples: <span>, <a>, <strong>, <em>, <img>
  • They can only have padding and margin on left and right (vertical margins/paddings may not affect layout).
  • They do not cause line breaks.

Example of Inline elements:

<p>This is a paragraph with <strong>strong</strong> and <em>emphasized</em> text.</p>
<span style="color: #d6006f;">This text is inside a span.</span>

This is a paragraph with strong and emphasized text.

This text is inside a span.
