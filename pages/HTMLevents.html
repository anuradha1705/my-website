HTML Events

HTML Events are actions that occur in the browser, and you can respond to them using JavaScript. Events happen when users interact with a webpage — like clicking, hovering, typing, loading, etc.

Common HTML Events

  • onclick – when a user clicks on an element
  • onmouseover – when a user hovers over an element
  • onchange – when an input value changes
  • onload – when the page finishes loading
  • onkeydown – when a key is pressed down
  • onsubmit – when a form is submitted

Syntax

Events are written in HTML using the attribute format:

<element event="JavaScriptCode">

Example: Click Event

<button onclick="showMessage()">Click Me!</button>

<script>
  function showMessage() {
    alert("You clicked the button!");
  }
</script>

Output:

Explanation:

  • The onclick attribute is added inside the button tag.
  • When the user clicks the button, the function alert() shows a popup.
Summary:

Events let you add interactivity to your HTML elements. You can respond to user actions like clicking, typing, loading, etc., using JavaScript. The most commonly used event is onclick, and it helps make websites more dynamic and interactive.
