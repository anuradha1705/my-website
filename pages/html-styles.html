HTML Styles

In HTML, the style attribute is used to add CSS (Cascading Style Sheets) directly inside an element. It helps to change how an element looks — like its color, size, font, and more.

Syntax:

<element style="property:value;">Content</element>

Example:

<p style="color:blue; font-size:20px;">
This is a styled paragraph.
</p>

<h1 style="color:red; text-align:center;">
Styled Heading
</h1>

Explanation:

  • color:blue; changes text color to blue.
  • font-size:20px; increases text size.
  • text-align:center; centers the heading.
  • Multiple properties can be added by separating them with semicolons (;).

Common Style Properties:

  • color – changes text color
  • background-color – sets background color
  • font-size – sets size of the text
  • text-align – aligns text (left, right, center)
  • font-family – changes font style

Note:

  • Use inline styles only for small changes.
  • For large styling, use internal or external CSS (which you’ll learn next).
