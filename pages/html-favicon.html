HTML Favicon

A Favicon is the small icon you see in the browser tab next to the page title. It's used to help identify the website visually.

How to Add a Favicon

Favicons are added using the <link> tag inside the <head> section of your HTML page.

<head>
  <link rel="icon" type="image/x-icon" href="favicon.ico">
</head>

Example

If your favicon.ico file is in the root folder of your project, it will appear automatically in the browser tab.

Common Favicon File Types

  • .ico (most common and recommended)
  • .png, .jpg, .svg (modern browsers support these too)

Tips

  • Favicon image size is usually 16x16 pixels or 32x32 pixels.
  • Save the favicon file in the root directory or mention the correct path.
  • Browsers may cache favicons, so changes might not show immediately.
