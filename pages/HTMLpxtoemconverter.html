PX to EM Converter

In web design, px (pixels) and em (relative units) are used for font sizing and spacing. 1em is equal to the current font size. If the base font size is 16px, then:

1em = 16px by default

Formula

EM = Pixel Value ÷ Base Font Size
(Example: 32px ÷ 16 = 2em)
