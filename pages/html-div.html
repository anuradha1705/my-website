The HTML <div> Element

The <div> element is a block-level container used to group other HTML elements together. It is very useful for applying CSS styles or JavaScript to multiple elements as a single unit.

What is <div>?

- The <div> tag doesn't inherently represent anything on its own, but is used to organize content.
- It is a block-level element, meaning it starts on a new line and stretches the full width available.
- You can add CSS styles and IDs/classes to <div> elements to customize layout and design.

Syntax:

<div>
  ...content...
</div>

Example Usage:

<div style="background-color: #ffccdd; padding: 15px; border-radius: 10px;">
  <h2>Welcome to my website</h2>
  <p>This is inside a <div> container.</p>
</div>

Output:

Welcome to my website

This is inside a <div> container.

Why use <div>?

  • Group related elements to apply CSS styling.
  • Use as layout blocks in page design.
  • Manipulate groups of elements with JavaScript.
  • Essential for creating sections in modern web pages.
