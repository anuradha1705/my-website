HTML Form Attributes

The <form> element uses various attributes to control form submission and behavior.

Key Form Attributes

  • action: URL where form data is sent on submit.
  • method: HTTP method used (GET or POST).
  • enctype: Encoding type for form data, especially important for file uploads.
  • target: Where to open the response (_self, _blank, etc.).
  • name: Name of the form for scripting.
  • novalidate: Disable HTML5 validation.
  • autocomplete: Enable/disable autofill (on or off).

Example: Using Multiple Attributes in a Form

<form action="submit.php" method="post" enctype="multipart/form-data" target="_blank" autocomplete="off">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" placeholder="Enter username" required />

  <label for="profilepic">Upload Profile Picture:</label>
  <input type="file" id="profilepic" name="profilepic" />

  <input type="submit" value="Register" />
</form>
