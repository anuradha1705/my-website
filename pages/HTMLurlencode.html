HTML URL Encoding

URL Encoding (also called Percent Encoding) is used when placing data into a URL, especially form data or special characters.

Why URL Encoding?

  • URLs can only be sent over the Internet using the ASCII character set.
  • Characters like spaces, &, %, ?, # need to be encoded to be used in URLs.

Common Encoded Characters

  • Space → %20
  • & → %26
  • = → %3D
  • ? → %3F
  • # → %23

Syntax Example

Original URL:
https://www.example.com/search?query=flowers & trees

After Encoding:
https://www.example.com/search?query=flowers%20%26%20trees

Form Example

<form action="/search">
  <input type="text" name="q" value="flowers & trees">
  <input type="submit">
</form>

Output

Summary:

URL encoding is essential when using special characters in URLs. Always encode data like spaces, ampersands (&), and others using proper percent encoding to avoid errors and maintain clean URLs.
