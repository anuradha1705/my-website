HTML Exercises

Practice these basic HTML exercises to improve your skills. Try writing the code on your own before checking the solution.

Exercise 1: Basic Page Structure

Create a simple HTML page with a title and heading.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Exercise 2: Paragraph and Line Break

Create a paragraph and use a line break inside it.

<p>
  This is line one.<br>
  This is line two.
</p>

Exercise 3: Image and Link

Display an image and a clickable link below it.

<img src="logo.png" alt="Logo" width="200">
<br>
<a href="https://example.com">Visit Example</a>

Exercise 4: Create a List

Make an unordered list of your favorite fruits.

<ul>
  <li>Mango</li>
  <li>Apple</li>
  <li>Banana</li>
</ul>

Exercise 5: Table Structure

Create a table with two rows and three columns.

<table border="1">
  <tr>
    <td>A1</td>
    <td>A2</td>
    <td>A3</td>
  </tr>
  <tr>
    <td>B1</td>
    <td>B2</td>
    <td>B3</td>
  </tr>
</table>
