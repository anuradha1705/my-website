HTML Forms

HTML forms allow users to enter data and send it to a server for processing. Forms are essential for tasks like login, registration, surveys, and feedback.

Form Tag & Attributes

The <form> tag creates a form. Important attributes include:

  • action: URL where form data is sent after submission.
  • method: HTTP method to send data. Usually GET or POST.
  • enctype: Encoding type for form data (important for file uploads).

Common Form Elements

  • <input>: Single-line input field (text, password, checkbox, radio, submit, etc.)
  • <textarea>: Multi-line text input.
  • <select> and <option>: Dropdown lists.
  • <button>: Buttons to submit or reset forms.
  • <label>: Labels for form elements (improves accessibility).

Basic Form Syntax

<form action="submit.php" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required>
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <br>
  <input type="submit" value="Submit">
</form>
