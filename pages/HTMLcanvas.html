HTML <canvas> Tag

The <canvas> element in HTML is used to draw graphics like lines, shapes, or even games using JavaScript. It creates a drawing area that can be styled and scripted.

Syntax

<canvas id="myCanvas" width="300" height="150">
  Your browser does not support the canvas tag.
</canvas>

Example with Output

<canvas id="myCanvas" width="300" height="150"></canvas>

<script>
  const canvas = document.getElementById("myCanvas");
  const ctx = canvas.getContext("2d");
  ctx.fillStyle = "#ff4081";
  ctx.fillRect(40, 30, 200, 80);
</script>

Output:

Explanation

  • <canvas> creates the drawing area with width and height.
  • getContext("2d") is used to draw in 2D mode.
  • fillStyle sets the color of shapes.
  • fillRect(x, y, width, height) draws a filled rectangle.
Summary:

HTML Canvas is a powerful tag for drawing graphics using JavaScript. You can draw shapes, charts, animations, and more using the 2D or WebGL context.
