HTML Headings

HTML provides 6 levels of headings using the <h1> to <h6> tags. Headings help define structure and organize content clearly for users and search engines.

Heading Tags:

The six tags are:

  • <h1> - Main heading (most important)
  • <h2> - Sub-heading
  • <h3> - Sub-sub-heading
  • <h4> - Smaller heading
  • <h5> - Smaller heading
  • <h6> - Smallest heading

Example:

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
<h4>This is Heading 4</h4>
<h5>This is Heading 5</h5>
<h6>This is Heading 6</h6>

This is Heading 1

This is Heading 2

This is Heading 3

This is Heading 4

This is Heading 5
This is Heading 6

Tips:

  • Use only one <h1> per page for SEO.
  • Use headings in proper order like h1 → h2 → h3.
  • Do not use them just to make text bold — they are for structure!
