HTML JavaScript

JavaScript is a programming language used to add interactivity, behavior, and dynamic content to HTML pages. It runs in the browser and can manipulate HTML elements, handle events, validate forms, create animations, and much more.

How to Add JavaScript to HTML?

There are 3 common ways to include JavaScript in an HTML page:

  • Inline using the <script> tag inside HTML.
  • Linking an external JavaScript file using <script src="filename.js"></script>.
  • Using event handler attributes directly in HTML tags (like onclick).

Basic Syntax:

<script>
  alert('Hello, World!');
</script>

Example 1: Inline JavaScript

<!DOCTYPE html>
<html>
<head>
  <title>JavaScript Example</title>
</head>
<body>
  <h2>Click the button</h2>
  <button onclick="showMessage()">Click me</button>

  <script>
    function showMessage() {
      alert('Hello from JavaScript!');
    }
  </script>
</body>
</html>

Output:

Click the button

Example 2: External JavaScript File

You can create a separate script.js file and link it like this:

<script src="script.js"></script>

And inside script.js:

function greet() {
  alert('Greetings from external JS file!');
}

Example 3: Using JavaScript to Change HTML Content

<!DOCTYPE html>
<html>
<body>
  <p id="demo">This is a paragraph.</p>
  <button onclick="changeText()">Change Text</button>

  <script>
    function changeText() {
      document.getElementById('demo').innerHTML = 'Text has been changed!';
    }
  </script>
</body>
</html>

Output:

This is a paragraph.
