HTML Lists

HTML Lists are used to group related items in a structured and readable format. There are three main types of lists in HTML:

  • Ordered List - Uses numbers (1, 2, 3...)
  • Unordered List - Uses bullets (‚óè)
  • Description List - Uses terms and their descriptions

1. Ordered List (<ol>)

<ol>
  <li>Apple</li>
  <li>Banana</li>
  <li>Mango</li>
</ol>
  1. Apple
  2. Banana
  3. Mango

2. Unordered List (<ul>)

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript

3. Description List (<dl>)

<dl>
  <dt>HTML</dt>
  <dd>Standard markup language for web pages</dd>
  <dt>CSS</dt>
  <dd>Used for styling HTML content</dd>
</dl>
HTML
Standard markup language for web pages
CSS
Used for styling HTML content
