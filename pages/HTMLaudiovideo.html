HTML Audio and Video

HTML allows you to embed audio and video files directly into web pages using the <audio> and <video> elements. These tags support multiple file formats and playback controls.

Audio Tag Syntax

<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Output:

Video Tag Syntax

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Output:

Explanation

  • <audio> is used to embed sound or music files.
  • <video> embeds videos and supports controls like play, pause, volume.
  • controls adds default play/pause UI.
  • <source> specifies file and type, helps browser choose best format.
Summary:

Use <audio> and <video> to add multimedia to websites. These tags are user-friendly, responsive, and allow playback controls for better experience.
