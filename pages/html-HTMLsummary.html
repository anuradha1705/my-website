HTML Summary

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It provides the structure of a webpage by using various elements called tags.

Key Points of HTML

  • Tags: HTML uses tags enclosed in angle brackets <> to define elements.

  • Elements: Basic building blocks such as headings, paragraphs, images, links, lists, etc.

  • Attributes: Provide additional information about elements, e.g., href in anchor tags.


  • Document Structure: Starts with <!DOCTYPE html>, followed by <html>, <head>, and


    <body>.


  • Semantic Tags: Tags like <header>, <footer>, <article>, which give meaning to the content.

  • Forms and Inputs: Used to collect user data.

  • Multimedia: Adding images, videos, audio, and interactive elements.

Basic HTML Document Syntax

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

Example: Simple HTML Page

This example shows a simple HTML page with headings and paragraphs:

<!DOCTYPE html>
<html>
<head>
  <title>My First Webpage</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is my first webpage made with HTML.</p>
</body>
</html>

Summary

HTML is the foundation of web development. It defines the structure and content of web pages through tags and elements. Learning HTML is essential to creating websites and understanding how the web works.

Remember:
  • HTML stands for HyperText Markup Language.
  • Tags define elements on the page.
  • Attributes provide extra details.
  • Semantic tags improve accessibility and SEO.


  • The basic HTML document structure must include <!DOCTYPE>, <html>, <head>, and


    <body>.
