HTML Drag and Drop

Drag and Drop is a feature that allows users to drag an element from one place and drop it to another on a web page. It enhances interactivity and user experience.

üîç How Drag and Drop works

To enable drag and drop in HTML and JavaScript, you typically:

  • Make an element draggable by setting draggable="true".
  • Handle drag events like ondragstart to store the dragged data.
  • Set the drop target to allow dropping by preventing the default behavior using ondragover.
  • Handle the ondrop event to specify what happens when an item is dropped.

<!-- Draggable element -->
<div id="drag1" draggable="true" ondragstart="drag(event)">Drag me!</div>

<!-- Drop target -->
<div id="div1" ondrop="drop(event)" ondragover="allowDrop(event)">
  Drop here
</div>

<script>
function allowDrop(ev) {
  ev.preventDefault();
}

function drag(ev) {
  ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
  ev.preventDefault();
  var data = ev.dataTransfer.getData("text");
  ev.target.appendChild(document.getElementById(data));
}
</script>

üìù Explanation

  • draggable="true" makes an element draggable.
  • ondragstart fires when dragging starts; data is stored with dataTransfer.setData.
  • ondragover allows dropping by preventing default behavior.
  • ondrop handles the drop, appending the dragged element to the drop target.
  • Data is transferred using a key-value pair; here key is "text".
