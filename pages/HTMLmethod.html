HTML method Attribute

The method attribute in HTML is used inside the <form> tag to define how form data should be sent to the server.

Types of Methods

  • GET – Sends form data in the URL (visible in address bar). Used for small, non-sensitive data.
  • POST – Sends data in the background (invisible). Used for secure or large data.

Syntax

<form action="submit.php" method="get">
  ...
</form>

<form action="submit.php" method="post">
  ...
</form>

Example

<form action="#" method="get">
  Name: <input type="text" name="name"><br><br>
  <input type="submit" value="Submit using GET">
</form>

<br><hr><br>

<form action="#" method="post">
  Name: <input type="text" name="name"><br><br>
  <input type="submit" value="Submit using POST">
</form>

Summary:

The method attribute is used in the <form> tag to decide how form data will be sent. Use GET for non-sensitive data (like searches), and POST for secure or private data (like login forms).
