HTML Global Attributes

Global attributes are HTML attributes that can be used on all HTML elements. They help control how elements behave, appear, and interact with users.

Some Common Global Attributes:

  • class – Assigns a class name to the element for styling.
  • id – Assigns a unique ID to the element.
  • style – Adds inline CSS styles directly to the element.
  • title – Adds tooltip text that appears on hover.
  • hidden – Hides the element from the page.
  • contenteditable – Makes the content inside editable.

Example:

<p class="pink-text" id="myPara" style="color:hotpink;" title="Hover text">
  This is a paragraph with global attributes.
</p>

This is a paragraph with global attributes.

Explanation:

  • class="pink-text" — helps apply CSS styles.


  • id="myPara" — uniquely identifies this paragraph.


  • style="color:hotpink;" — directly colors the text.


  • title="Hover text" — shows a small box when hovered.


Summary:

Global attributes work on almost all HTML tags and help improve control and interactivity. Examples include

class, id, style, title, hidden, and contenteditable. These are useful for


styling, scripting, and accessibility.
