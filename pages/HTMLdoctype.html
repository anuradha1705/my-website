HTML DOCTYPE Declaration

The DOCTYPE declaration in HTML tells the web browser what version of HTML the page is written in. It should always be the very first line in your HTML document — even before the <html> tag.

Syntax

<!DOCTYPE html>

Example

<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>Welcome!</h1>
  </body>
</html>

Explanation

  • <!DOCTYPE html> defines the document as HTML5.
  • It helps the browser render the page correctly and consistently.
  • It is *not* an HTML tag, but an instruction to the browser.
  • There is no closing tag for DOCTYPE.
Summary:

The DOCTYPE declaration is a simple but important part of every HTML document. It tells browsers to use the latest HTML5 rendering mode. Always use <!DOCTYPE html> at the top of your page.
