HTML Example Page

This page shows a simple but complete HTML document. It helps beginners understand the basic structure of an HTML page, including elements like <!DOCTYPE>, <html>, <head>, <body>, etc.

üìÑ Complete HTML Syntax

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to DevByAnu!</h1>
    <p>This is a paragraph of text.</p>
    <p>HTML is fun to learn!</p>
  </body>
</html>

Output

Welcome to DevByAnu!

This is a paragraph of text.

HTML is fun to learn!

üìå Explanation

  • <!DOCTYPE html>: Tells the browser this is an HTML5 document.
  • <html>: Root element of the HTML page.
  • <head>: Contains meta info like title, styles, etc.
  • <title>: Sets the title of the web page shown in the browser tab.
  • <body>: Everything visible on the web page goes inside this.
  • <h1>: Heading tag for main heading.
  • <p>: Paragraph tag to display text.
