HTML Interview Preparation

1. What is HTML?

Answer:

HTML (HyperText Markup Language) is the standard markup language used to create and design pages on the web. It structures the content using tags and elements.

2. What are the basic components of an HTML document?

Answer:

An HTML document consists of <!DOCTYPE html>, <html>, <head>, and <body> tags. The head contains metadata and title, while body contains the visible content.

3. Explain the difference between id and class attributes.

Answer:

The id attribute uniquely identifies a single element and must be unique within the page. The class attribute can be used to group multiple elements for styling or scripting purposes.

4. What is semantic HTML?

Answer:

Semantic HTML uses meaningful tags to structure web pages, such as <header>, <nav>, <section>, <article>, and <footer>. This improves accessibility and SEO.

5. How do you include CSS and JavaScript in an HTML page?

Answer:

CSS can be included inline, internally using <style> tags inside <head>, or externally via <link> tags. JavaScript can be added inline, internally with <script> tags, or externally via script files.

6. What is the purpose of the alt attribute in <img> tags?

Answer:

The alt attribute provides alternative text for an image if it cannot be displayed. It improves accessibility for screen readers and SEO.

7. Write a simple HTML code to create a hyperlink.

Answer:

<a href="https://www.example.com">Visit Example</a>

8. How do you create a list in HTML? Explain with examples.

Answer:

HTML supports ordered lists (<ol>), unordered lists (<ul>), and description lists (<dl>). Example: 

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

9. What are void elements in HTML?

Answer:

Void elements are self-closing tags that do not have closing tags, such as <br>, <img>, <input>, and <hr>.

10. How can you make your HTML page responsive?

Answer:

Use the viewport meta tag <meta name="viewport" content="width=device-width, initial-scale=1"> and CSS media queries to adapt the layout for different screen sizes.
