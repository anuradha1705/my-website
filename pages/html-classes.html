HTML Classes

The class attribute in HTML allows you to assign one or more class names to an element. Classes are used to group elements for applying CSS styles or manipulating them with JavaScript.

What is the class attribute?

- It provides a way to classify elements into groups.
- Multiple elements can share the same class.
- You can assign multiple classes to a single element separated by spaces.
- It is commonly used with CSS selectors and JavaScript DOM manipulation.

Syntax:

<tag class="className1 className2">...</tag>

Example:

<style>
  .highlight {
    background-color: yellow;
    font-weight: bold;
  }
</style>

<p class="highlight">This paragraph is highlighted.</p>
<p>This paragraph is normal.</p>

Output:

This paragraph is highlighted.

This paragraph is normal.

Using Multiple Classes:

<style>
  .bold { font-weight: bold; }
  .red { color: red; }
</style>

<p class="bold red">This text is bold and red.</p>

Output:

This text is bold and red.

Why use Classes?

  • Apply the same style to multiple elements without repeating CSS.
  • Target specific groups of elements with JavaScript easily.
  • Make your HTML more semantic and organized.
