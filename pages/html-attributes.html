HTML Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

Basic Syntax

<element attribute="value">Content</element>

Common HTML Attributes

  • href - Specifies the URL for a link (used in <a>)
  • src - Specifies the image path (used in <img>)
  • alt - Alternative text for images
  • title - Provides extra info when hovered
  • style - Inline CSS styling
  • id and class - Used to uniquely identify and group elements
  • target - Specifies where to open linked document (e.g. _blank)

Examples

<a href="https://www.example.com" target="_blank">Visit Example</a>

<img src="car.jpg" alt="Car Image" width="300" height="200">

<p title="I'm a tooltip">Hover over me!</p>

<h2 id="main-heading" style="color: blue;">Styled Heading</h2>

Important Notes

  • Always use quotes around attribute values.
  • Attributes should be written in lowercase.
  • Order of attributes doesn't matter.
  • Use meaningful alt text for better accessibility.
