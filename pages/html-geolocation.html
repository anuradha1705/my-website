HTML Geolocation API

Geolocation API is used to get the geographical location (latitude and longitude) of the user's device. This is commonly used in location-based apps like maps, delivery tracking, etc.

üìå Syntax:

navigator.geolocation.getCurrentPosition(successFunction);

üìç Example Code:

<button onclick="getLocation()">Get My Location</button> <p id="location"></p> <script> function getLocation() { if (navigator.geolocation) { navigator.geolocation.getCurrentPosition(showPosition); } else { document.getElementById("location").innerHTML = "Geolocation not supported."; } } function showPosition(position) { document.getElementById("location").innerHTML = "Latitude: " + position.coords.latitude + "<br>Longitude: " + position.coords.longitude; } </script>

üìí Explanation:

  • navigator.geolocation: Checks if browser supports geolocation.
  • getCurrentPosition(): Asks for user permission and gets coordinates.
  • showPosition(): Displays the latitude and longitude.
  • Error handling is shown in showError().

‚úÖ Summary

  • Geolocation API helps access user's current location (lat/lon).
  • Needs user permission before accessing location data.
  • Useful in apps like Google Maps, delivery, ride sharing, etc.
  • Not supported on all browsers/devices without permission.
