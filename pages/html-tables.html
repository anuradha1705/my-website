HTML Tables

HTML tables allow you to arrange data into rows and columns, making it easier to read and understand structured information.

Basic Table Syntax

<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Example with Output

Here's a simple table showing student marks:

<table>
  <tr>
    <th>Name</th>
    <th>Marks</th>
  </tr>
  <tr>
    <td>Anu</td>
    <td>95</td>
  </tr>
  <tr>
    <td>Aaradhya</td>
    <td>88</td>
  </tr>
</table>

Output:

Name Marks
Anu 95
Aaradhya 88
