HTML with CSS

CSS (Cascading Style Sheets) is used to style and design HTML content. It helps you change colors, layout, fonts, borders, spacing, and much more to make your webpages beautiful and user-friendly.

Ways to Apply CSS

  • Inline CSS: CSS directly written inside an HTML tag using the style attribute.
  • Internal CSS: CSS inside a <style> tag in the <head> of the HTML document.
  • External CSS: CSS written in a separate file with .css extension, linked using <link> tag.

Syntax Example

<p style="color: purple; font-size: 20px;">Styled with Inline CSS</p> <style>
h1 {
color: hotpink;
text-align: center;
}
</style>
