HTML Colors

Colors add life to your webpages. HTML allows you to set colors for text, backgrounds, borders, and other elements using different methods.

Color Formats in HTML

Common ways to specify colors in HTML and CSS are:

  • Color Names: Using predefined color names like red, blue, green, etc.
  • Hexadecimal (Hex) Codes: Using a 6-digit hex code like #ff0000 for red.
  • RGB Values: Using RGB function with red, green, blue values from 0 to 255, e.g., rgb(255, 0, 0).
  • RGBA Values: Like RGB but with an alpha transparency value, e.g., rgba(255, 0, 0, 0.5).

Syntax to Set Color

You typically set colors with CSS styles. For example:

<p style="color: red;">This text is red.</p>

<div style="background-color: #00ff00;">Green background</div>

Example: Different Color Formats

<h3 style="color: blue;">This is blue text</h3>
<p style="color: #ff6600;">This text uses hex color code</p>
<div style="background-color: rgb(0, 255, 255);">This div has cyan background</div>
<span style="color: rgba(255, 0, 0, 0.7);">This text is semi-transparent red</span>

Color Names vs Hex Codes

Color names are easy to remember but limited in number.

Hex codes provide millions of colors for precise control.

RGB and RGBA add even more flexibility, especially for transparency.
